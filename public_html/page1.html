Let's Kick Things Off

The definition of the internet can best be described as a collection of networks. It is how a network set up in one part of the world, can communicate with a network set up all the way in another end of the world. It has revolutionized communication, collaboration, innovation, and culture. Today with the internet, location becomes oblivious as virtually anybody from anywhere can broadcast their messages and thought across tremendous lengths.
